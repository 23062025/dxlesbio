Python'a Giriş: İlk Adımlarınız+

Python, basit ve okunabilir sözdizimi sayesinde yeni başlayanlar için harika bir dildir. Veri bilimi, yapay zekâ, web geliştirme ve otomasyon gibi birçok alanda kullanılır. En büyük avantajı, kod yazmayı bir metin yazmak kadar kolay hale getirmesidir.

1. Ekrana Yazı Yazmak: `print()` ve `import`

Python'da ekrana bir mesaj yazdırmak için **print()** fonksiyonunu kullanırız. Tırnak işaretleri, bunun bir metin (`string`) olduğunu gösterir. **import** kelimesi ise başka birinin yazdığı hazır kodları veya fonksiyonları projenize dahil etmenizi sağlar. Bu sayede tekerleği yeniden icat etmek zorunda kalmazsınız. 

import math # Matematik kütüphanesini dahil ettik
print("Merhaba Dünya!")
print("Pi sayısı:", math.pi)

2. Değişkenler: Bilgiyi Saklamak

Değişkenler, verileri sakladığımız kutucuklardır. Python'da bir değişkenin tipini (sayı, metin, vs.) belirtmenize gerek yoktur.

  • **String (Metin):** isim = "DXLES"
  • **Integer (Tam sayı):** yas = 15
  • **Float (Ondalıklı sayı):** boy = 1.75

Listeler, demetler ve sözlükler gibi karmaşık veri tipleri de mevcuttur.

  • **Listeler (`list`):** Sıralı ve değiştirilebilir koleksiyonlar. meyveler = ["elma", "armut", "muz"]
  • **Sözlükler (`dict`):** Anahtar-değer çiftlerinden oluşan yapılar. kullanici = {"ad": "DXLES", "rol": "Geliştirici"}

3. Koşullu İfadeler: `if`, `elif`, `else`

Programlarımızın farklı durumlara göre farklı şeyler yapmasını sağlamak için koşullu ifadeleri kullanırız. 

yas = 20

if yas >= 18:
    print("Reşitsiniz.")
else:
    print("Reşit değilsiniz.")

notu = 85
if notu > 90:
    print("Harika!")
elif notu > 70:
    print("İyi.")
else:
    print("Tekrar denemelisin.")

4. Döngüler: `for` ve `while`

Belirli bir işlemi defalarca tekrarlamak için döngüleri kullanırız.

  • **`for` Döngüsü:** Belirli bir aralıktaki her eleman için kodu çalıştırır.
  • **`while` Döngüsü:** Belirli bir koşul doğru olduğu sürece kodu çalıştırır.
# 1'den 5'e kadar sayıları ekrana yazdırır (for)
for i in range(1, 6):
    print(i)

# 10'dan geriye sayar (while)
sayi = 10
while sayi > 0:
    print(sayi)
    sayi -= 1
Java'ya Giriş: Nesne Yönelimli Programlama+

Java, nesne yönelimli (Object-Oriented) bir programlama dilidir. "Bir kez yaz, her yerde çalıştır" sloganıyla bilinir, çünkü Java Virtual Machine (JVM) sayesinde her işletim sisteminde sorunsuz çalışır. Mobil (Android), kurumsal ve masaüstü uygulamalar için sıklıkla tercih edilir.

1. Sınıflar (`Class`) ve Nesneler (`Object`)

Java'da her şey bir sınıf içinde tanımlanır. Sınıflar, nesnelerin şablonlarıdır. Bir sınıf, nesnelerin sahip olabileceği özellikleri (değişkenler) ve davranışları (metotlar) tanımlar. 

public class Araba {
    String renk; // Özellik (değişken)
    int hiz;     // Özellik (değişken)

    public void gazVer() { // Davranış (metot)
        System.out.println("Araba hızlanıyor.");
    }
}

Bir sınıfı tanımladıktan sonra, ondan nesneler oluşturabiliriz: 

Araba benimArabam = new Araba();
benimArabam.renk = "Kırmızı";
benimArabam.gazVer();

2. Ana Metot (`main`) ve Çıktı

Her Java programı, **public static void main(String[] args)** ana metodu ile başlar. Bu, programın giriş noktasıdır.

* **System.out.println():** Ekrana bir satır metin yazar ve imleci bir alt satıra atar. * **System.out.print():** Ekrana bir metin yazar, ancak alt satıra geçmez.

3. Değişkenler ve Döngüler

Java'da değişken tanımlarken veri tipini belirtmek zorunludur. 

// int, double, boolean gibi temel tipler
int yas = 20;
double boy = 1.80;
boolean isOgrenci = true;

// 1'den 10'a kadar sayıları toplayan döngü
int toplam = 0;
for (int i = 1; i <= 10; i++) {
    toplam = toplam + i;
}
System.out.println("Toplam: " + toplam);
HTML ve CSS'e Giriş: Web Geliştirmenin Temeli+

HTML ve CSS, birbirini tamamlayan iki temel web teknolojisidir.

**HTML (HyperText Markup Language):** Web sayfasının iskeletini oluşturur. Hangi başlık, hangi paragraf, nerede bir resim olacağı gibi içerik yapısını belirler.

**CSS (Cascading Style Sheets):** Sayfanın görselini belirler. Renkler, yazı tipleri, boyutlar, boşluklar ve animasyonlar gibi tüm stil ayarları CSS ile yapılır.

1. Temel HTML Etiketleri

HTML'de içerikler **etiketler** (`tags`) arasına yazılır. 

<!-- Başlık ve Paragraf -->
<h1>HTML Öğreniyorum</h1>
<p>Bu bir paragraf etiketidir.</p>

<!-- Liste -->
<ul>
  <li>Kahve iç</li>
  <li>Kod yaz</li>
</ul>

<!-- Link ve Resim -->
<a href="https://www.google.com">Google'a Git</a>
<img src="resim.jpg" alt="Harika bir resim">

<!-- Basit bir form -->
<form>
  <label for="ad">Adın:</label>
  <input type="text" id="ad">
  <button type="submit">Gönder</button>
</form>

Hakkımda (Soru-Cevap)

Soru: DXLES kimdir?+
Dxles 2018’de yazılım geliştirmeye başlayan, Almanya merkezli genç bir full-stack yazılımcıdır. Çalışmalarında hem frontend hem de backend alanlarında projeler üretmiş; web siteleri, botlar, RAT sistemleri, SQL tabanlı yapılar ve farklı arayüz konseptleri geliştirmiştir. Özellikle siber güvenlik, SQL injection araştırmaları ve uzaktan kontrol yazılımları üzerine yoğunlaşmış, Telegram/Discord tabanlı kontrol sistemleri kurmuştur. Teknik becerilerinin yanında görselliğe ve tarza da önem vermiş, “illegal software” havası taşıyan site tasarımları ve özgün arayüzlerle dikkat çekmiştir. Bunun dışında Instagram üzerinde hesap kapatma işleri, banned geçmişi ve alaycı içerikleriyle de bilinir. Kısaca Dxles, genç yaşta yazılıma başlayan, saldırı ve savunma tarafında kendini geliştiren, farklı projeleriyle öne çıkan üretken bir yazılımcıdır..
Soru: Hangi teknolojilerle çalışıyorsun?+
Frontend tarafında HTML, CSS ve JavaScript ile çalışıyorum; modern arayüz tasarımlarında farklı kütüphaneleri de kullanıyorum. Backend tarafında ise Python, Node.js ve PHP üzerine yoğunlaştım. SQL veritabanlarıyla veri çekme, işleme ve site entegrasyonu konusunda deneyimim var. Ayrıca C#, C++ ve Java gibi dillerde de temel seviyede bilgim bulunuyor.Bunların yanında Telegram ve Discord bot geliştirme, RAT (Remote Access Tool) sistemleri kurma, yapay zekâ entegrasyonu ve güvenlik araştırmaları gibi alanlarla da ilgileniyorum. Tasarım tarafında ise daha çok “illegal software” havası veren özgün arayüzler, neon temalar ve kullanıcı deneyimine yönelik özel konseptler oluşturuyorum.
Soru: Gelecekteki projelerin neler?+
Gelecekteki projelerim arasında daha gelişmiş RAT sistemleri ve Telegram/Discord tabanlı kontrol panelleri oluşturmak var. Bunun yanında SQL tabanlı sistemleri daha derinlemesine kullanarak kendi veri platformlarımı kurmayı, API’ler üzerinden otomatik çalışan siteler geliştirmeyi ve farklı güvenlik açıklarını test ederek eğitim amaçlı araçlar çıkarmayı planlıyorum. Ayrıca web tasarım tarafında “illegal software” estetiğini daha ileriye taşımak, hem görsel hem de fonksiyonel açıdan dikkat çeken arayüzler üretmek istiyorum. Bununla birlikte yapay zekâ entegrasyonu, kendi oyun denemelerim ve kullanıcıya farklı deneyimler sunacak özel yazılımlar da gelecekteki planlarım arasında.
Soru: Bu web sitesi neden böyle bir tasarıma sahip?+
Bu site bu tasarıma sahip çünkü Dxles dijital kimliğini, ilgi alanlarını ve tarzını yansıtmak istiyor — siberpunk / bilim kurgu estetiği, “fütüristik” bir atmosfer, karanlık gölgeler, neon grid efektleri gibi görsel ögelerle siteyi sıradan olmaktan çıkarıyor.
Soru: Yapay zekâya ilgin nereden geliyor?+
Yapay zekâya ilgim, makineleri sadece çalıştırmak değil, onlara öğrenme ve düşünme yeteneği kazandırma merakından geliyor. Botlar, otomatik sistemler ve güvenlik araçlarını daha akıllı hale getirmek beni cezbediyor.
Soru: Siber araştırmalar hakkında biraz daha bilgi verir misin?+
Siber araştırmalarım, güvenlik açıklarını keşfetmek, sistemleri test etmek ve farklı yazılımların çalışma mantığını anlamak üzerine. RAT’lar, SQL injection ve bot sistemleri gibi araçlarla hem saldırı hem savunma tarafını inceliyorum. Ama amaç hep öğrenmek ve kendi projelerimde kullanmak; yani keşfetmek ve deneyim kazanmak öncelikli.
Soru: Kendi projelerimi nasıl geliştirebilirim?+
Kendi projelerini geliştirmek için önce temel programlama bilgini sağlamlaştırmalısın. Eğitim kısmında Python, Java, C#, C++, HTML gibi dillerin temelini öğrenebilir, adım adım kodlama mantığını kavrayabilirsin. Daha sonra küçük prototipler yaparak fikirlerini test et, hatalardan öğren ve projelerini geliştirmeye devam et. Python'a Giriş: İlk Adımlarınız+

Python, basit ve okunabilir sözdizimi sayesinde yeni başlayanlar için harika bir dildir. Veri bilimi, yapay zekâ, web geliştirme ve otomasyon gibi birçok alanda kullanılır. En büyük avantajı, kod yazmayı bir metin yazmak kadar kolay hale getirmesidir.

**1. Ekrana Yazı Yazmak: `print()` Fonksiyonu**
Python'da ekrana bir mesaj yazdırmak için `print()` fonksiyonunu kullanırız. Parantez içine yazdığımız her şey ekrana aynen yazdırılır. Tırnak işaretleri, bunun bir metin (`string`) olduğunu gösterir. 

print("Merhaba Dünya!")
print("Adım DXLES.")
Merhaba Dünya! Adım DXLES.

**2. Değişkenler: Bilgiyi Saklamak**
Değişkenler, verileri sakladığımız kutucuklardır. Python'da bir değişkenin tipini (sayı, metin, vs.) belirtmenize gerek yoktur. 

# String (Metin) değişkeni
isim = "DXLES"

# Integer (Tam sayı) değişkeni
yas = 15

# Değişkenleri ekrana yazdırma
print("Kullanıcı Adı:", isim)
print("Yaşı:", yas)
Kullanıcı Adı: DXLES Yaşı: 15
Python: Veri Yapıları ve Fonksiyonlar+

Python'da verileri düzenlemek için çeşitli yapılar kullanılır. En yaygın olanları **Listeler** ve **Sözlükler**dir.

**1. Listeler (`list`)**
Listeler, birden fazla öğeyi sıralı olarak saklayan koleksiyonlardır. Köşeli parantez `[]` ile oluşturulur. 

# Liste tanımlama
meyveler = ["elma", "armut", "muz"]

# Bir öğeye erişim
print(meyveler[0]) # Çıktı: elma

# Yeni öğe ekleme
meyveler.append("çilek")
print(meyveler)
elma ['elma', 'armut', 'muz', 'çilek']

**2. Sözlükler (`dict`)**
Sözlükler, anahtar-değer çiftleri halinde verileri saklar. Süslü parantez `{}` ile oluşturulur. 

# Sözlük tanımlama
kullanici = {
  "isim": "DXLES",
  "yas": 15,
  "developer": True
}

# Bir değere erişim
print(kullanici["isim"])
DXLES

**3. Fonksiyonlar**
Fonksiyonlar, belirli bir görevi yerine getiren yeniden kullanılabilir kod bloklarıdır. `def` anahtar kelimesi ile tanımlanır. 

def selamlama(isim):
  print("Merhaba, " + isim + "!")

selamlama("Yazılımcı")

# Değer döndüren fonksiyon
def topla(a, b):
  return a + b

sonuc = topla(5, 3)
print("Sonuç:", sonuc)
Merhaba, Yazılımcı! Sonuç: 8
Java'ya Giriş: Temel Kavramlar+

Java, nesne yönelimli (Object-Oriented) bir programlama dilidir. "Bir kez yaz, her yerde çalıştır" sloganıyla bilinir, çünkü Java Virtual Machine (JVM) sayesinde her işletim sisteminde sorunsuz çalışır. Mobil (Android), kurumsal ve masaüstü uygulamalar için sıklıkla tercih edilir.

**1. Sınıflar ve Ana Metot**
Java'da her şey bir sınıf (`class`) içinde tanımlanır. Programın başlangıç noktası her zaman `main` metodudur. 

public class IlkProgramim {
    public static void main(String[] args) {
        System.out.println("Merhaba Dünya!");
    }
}
Merhaba Dünya!

**2. Değişkenler ve Veri Tipleri**
Java'da değişken tanımlarken veri tipini belirtmek zorunludur. 

// String (Metin)

// int (Tam sayı)

// double (Ondalıklı sayı)
double puan = 95.5;

System.out.println("Kullanıcı Adı: " + isim);
System.out.println("Yaşı: " + yas);

Java: Koşullu İfadeler ve Döngüler+

Java'da program akışını kontrol etmek için koşullu ifadeler (`if`, `else`) ve döngüler (`for`, `while`) kullanılır.

**1. Koşullu İfadeler**
`if` ve `else` blokları, belirli bir koşulun doğru olup olmadığına göre farklı kod bloklarını çalıştırmayı sağlar. 

int not = 75;

if (not >= 50) {
    System.out.println("Geçtiniz.");
} else {
    System.out.println("Kaldınız.");
}
Geçtiniz.

**2. `for` Döngüsü**
`for` döngüsü, belirli sayıda tekrarlamalar için kullanılır. 

for (int i = 1; i <= 5; i++) {
    System.out.println("Sayı: " + i);
}
Sayı: 1 Sayı: 2 Sayı: 3 Sayı: 4 Sayı: 5
Java: Nesneler ve Kalıtım+

Java, nesne yönelimli bir dil olduğu için programlar nesnelerden oluşur. Nesneler, sınıfların örnekleridir.

**1. Nesne Oluşturma**
Önce bir `Araba` sınıfı tanımlayalım ve ardından ondan bir nesne oluşturalım. 

public class Araba {
    String renk;
    int hiz;

    public Araba(String r, int h) {
        renk = r;
        hiz = h;
    }
}

// Main metot içinde nesne oluşturma
Araba bmw = new Araba("Siyah", 250);
System.out.println("Arabanın rengi: " + bmw.renk);
Arabanın rengi: Siyah

**2. Kalıtım (`Inheritance`)**
Kalıtım, bir sınıfın başka bir sınıftan özelliklerini ve metotlarını miras almasıdır. Bu, kod tekrarını önler. 

// Superclass (üst sınıf)
class Hayvan {
    void sesCikar() {
        System.out.println("Bilinmeyen bir ses.");
    }
}

// Subclass (alt sınıf)
class Kopek extends Hayvan {
    @Override
    void sesCikar() {
        System.out.println("Hav Hav!");
    }
}

// Main metot içinde kullanım
Kopek benimKopegim = new Kopek();
benimKopegim.sesCikar();
Hav Hav!

C++'a Giriş: Güç ve Performans+

C++, sistem programlama, oyun geliştirme, yüksek performans gerektiren uygulamalar ve gömülü sistemler gibi alanlarda kullanılan, çok güçlü ve hızlı bir dildir. C++'ı öğrenmek biraz daha zor olsa da, donanım üzerinde tam kontrol sağlar.

**1. Kütüphane Dahil Etme: `#include`**
C++'da bir fonksiyonu kullanmak için o fonksiyonun bulunduğu kütüphaneyi programa dahil etmemiz gerekir. `#include` komutu bunu sağlar. 

#include <iostream> // Giriş/çıkış işlemleri için

**2. Çıktı Verme: `std::cout`**
Ekrana metin yazdırmak için `iostream` kütüphanesinden gelen `std::cout` nesnesini kullanırız. `<<` operatörü ile yazdırmak istediğimiz veriyi `cout`'a aktarırız. 

#include <iostream>

int main() {
    std::cout << "Merhaba Dünya!";
    return 0;
}
Merhaba Dünya!

**3. Değişkenler**
Java gibi, C++'da da değişken tanımlarken tipini belirtmek zorunludur. 

#include <iostream>
#include <string> // Metin değişkeni için

int main() {
    // Tam sayı
    int yas = 25;

    // Metin
    std::string ad = "DXLES";

    std::cout << "Adınız: " << ad << std::endl;
    std::cout << "Yaşınız: " << yas << std::endl;
    return 0;
}
Adınız: DXLES Yaşınız: 25
C++: Diziler ve Fonksiyonlar+

C++'da verileri düzenlemek için diziler (`arrays`) kullanılır ve kod tekrarını önlemek için fonksiyonlar oluşturulur.

**1. Diziler**
Diziler, aynı veri tipindeki birden çok öğeyi sıralı olarak saklayan veri yapılarıdır. 

#include <iostream>

int main() {
    // 5 elemanlı bir tam sayı dizisi oluşturma
    int sayilar[5] = {10, 20, 30, 40, 50};

    // Dizinin 3. elemanını (indeks 2) ekrana yazdırma
    std::cout << "Dizinin 3. elemanı: " << sayilar[2] << std::endl;
    return 0;
}
Dizinin 3. elemanı: 30

**2. Fonksiyonlar**
C++'da bir fonksiyon, bir görevi yerine getiren ve programı daha düzenli hale getiren bir kod bloğudur. 

#include <iostream>

// Toplama işlemi yapan bir fonksiyon
int topla(int a, int b) {
    return a + b;
}

int main() {
    int sonuc = topla(15, 7);
    std::cout << "Toplam: " << sonuc << std::endl;
    return 0;
}
Toplam: 22
